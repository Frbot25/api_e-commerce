# A API REST WITH TYPESCRIPT, E-commerce

## Installation

### Install Dependencies

```javascript
npm i
```

### run

```javascript
npm run dev
```

view package.json

```json
    "dev": "tsc && node ./build/index.js",
    "start": "node ./build/index.js",
    "nodemon": "tsc && nodemon ./build/index.js",
    "test": "jest --watchAll --verbose",
    "type": "module"
```

Run test

```json
npm run test
```
